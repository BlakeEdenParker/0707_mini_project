# 가위바위보 이미지 분류 모델 학습기 (Rock-Paper-Scissors Classifier)

이 프로젝트는 텐서플로우(TensorFlow)와 케라스(Keras)를 활용하여 **'가위', '바위', '보' 이미지를 자동으로 분류하는 인공지능 모델**을 학습하고 생성하는 미니 프로젝트입니다.

## 📄 핵심 파일 소개: `index.ipynb`

`index.ipynb` 파일은 이 프로젝트의 가장 핵심적인 파일로, **이미지 데이터를 불러와 전처리하고, CNN 딥러닝 모델을 설계한 뒤 학습시켜 최종 모델 파일(`.h5`)을 뽑아내는 역할**을 수행합니다.

### 🎯 모델 학습의 목적
우리가 눈으로 가위, 바위, 보를 구분할 수 있듯, 인공지능이 주어진 손 모양 이미지를 보고 어떤 패를 냈는지 스스로 분류할 수 있도록 만드는 것이 목표입니다. 이 학습 과정을 통해 만들어진 모델은 이후 웹 애플리케이션(`app.py`) 등에 연동되어 사용자가 업로드한 이미지를 실시간으로 판별하는 데 사용됩니다.

---

## ⚙️ 동작 방식 및 파이프라인

`index.ipynb`는 다음과 같은 5단계의 파이프라인으로 동작합니다.

### 1. 데이터 수집 및 로드 (Data Loading)
- `data2` 폴더 내에 존재하는 `paper`, `rock`, `scissors` 3개의 하위 폴더를 탐색합니다.
- 각 폴더 안에 있는 모든 이미지 파일을 동적으로 긁어와서 파이썬 메모리 상에 불러옵니다.

### 2. 데이터 전처리 (Preprocessing)
- 불러온 이미지의 색상 모드를 표준 RGB로 맞춥니다.
- 연산 속도 향상과 모델 규격화를 위해 **모든 이미지를 가로세로 32x32 픽셀 크기로 일괄 축소(Resize)** 합니다.
- 픽셀 값(0~255)을 255.0으로 나누어 **0~1 사이의 값으로 정규화(Normalization)** 합니다. 이렇게 하면 인공지능이 패턴을 더 쉽고 빠르게 학습할 수 있습니다.

### 3. 학습 및 테스트 데이터 분할 (Data Split)
- 수집된 전체 이미지 데이터를 **학습용 데이터(Train) 80%**, **성능 검증용 테스트 데이터(Test) 20%** 비율로 나눕니다.
- 이때 각 클래스(가위/바위/보)의 비율이 한쪽으로 치우치지 않게 균등하게 분할(Stratify)합니다. 정답 라벨은 컴퓨터가 이해하기 쉬운 원-핫 인코딩(One-hot Encoding) 형식으로 변환합니다.

### 4. CNN 인공신경망 모델 구축 (Model Architecture)
- 시각적 이미지 패턴 인식에 특화된 **합성곱 신경망(CNN, Convolutional Neural Network)** 구조를 설계합니다.
- 총 3개의 합성곱 블록(Conv2D + BatchNormalization + ReLU + MaxPooling2D)을 거치며 이미지의 테두리, 질감, 형태 등의 특징을 추출해냅니다.
- 추출된 특징은 평탄화(Flatten)된 후, 일반적인 딥러닝 층(Dense)을 거치며 최종적으로 3가지(가위/바위/보) 확률 중 가장 높은 것을 선택하게 됩니다. 과적합 방지를 위해 Dropout 레이어도 포함되어 있습니다.

### 5. 모델 학습 및 저장 (Training & Saving)
- 설정된 구조를 바탕으로 `20 Epochs`(전체 데이터 반복 학습 20회) 동안 모델을 학습시킵니다.
- 학습이 성공적으로 완료되면 그 결과물을 **`rps_model.h5`** 라는 파일로 디스크에 저장합니다. 이 파일이 바로 우리 프로젝트의 완성된 "인공지능 뇌" 역할을 하게 됩니다.
